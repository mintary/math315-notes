\section{Heaviside function}
So far, we have dealt with so-called \textit{nice functions}. Consider the step function:
\[
u(t) =
\begin{cases}
0, \quad t < 0 \\
1, \quad t > 0
\end{cases}
\]

where $u(0)$ is undefined. This function is called the \textbf{Heaviside function}.

    \caption{Graph of the Heaviside function}
    \label{fig:heaviside}
\end{figure}

We can apply a shift to the Heaviside function such that it "turns on" at $t = a$:
\begin{figure}[H]

    \caption{Graph of the shifted Heaviside function}
    \label{fig:heaviside_shift}
\end{figure}

Finally, if we take two shifted Heaviside functions such that $a < b$, we get a function that resembles the following:
\begin{figure}[H]
    \centering
    \includegraphics[width=0.5\textwidth]{images/laplace_transforms/heaviside_function_subtract.png}
    \caption{Graph of the shifted Heaviside function}

We can see that this becomes useful to restrict the range of a function $f(t)$ to a certain interval. If the function is only "on" with $t \in (a, b)$, then we can write $f(t) = f(t)(u(t - a) - u(t - b))$.

\subsection{Bank account example}
Let's assume we have a bank account for which the change in our balance is modeled by the following:
\[
x(t + \Delta t) = x(t) + Ix(t) \Delta t
\]

where $x(t)$ is the amount current in the account, $\Delta t$ is some time over which interest is accumulated, and $I$ is the interest rate.

\textbf{Remark:} This discrete model is extremely similar to Euler's forward method where $f(x + h) = x + h \cdot f(x)$.

At $t = 0$, we have that $\dot{x} = Ix$. That is, the change in our balance is proportional to the amount currently in the account.
Let us define some rate $q(t)$ at which money is deposited into the account. Note that this is distinct from the interest rate $I$ which is constant. Then, we have that $\dot{x} - Ix = q(t)$.  We must now attempt to determine the cumulative total amount of money $Q(t)$ that has been deposite. It is clear that:
\[
\dot{Q} = q(t) \Longrightarrow Q(t) = \int_{0}^{t} q(\overline{t}) d\overline{t}
\]

Now consider the case where a single \$2000 occurs at $t = 1$, while we are also depositing money at a rate of \$1000 per year. 

Intuitively, we would write $Q(t)$ as:
\[
Q(t) = 1000t + 2000u(t - 1)
\]

We note that the Heaviside function is perfect to represent this situation: we "switch on" the deposit at $t = 1$ and the money remains in the account.

However, we are presented with the challenge of differentiating this function to derive $q(t)$. No such function exists for $u(t)$, but there is a "function" (which is actually a distribution) that satisifes the following properties that we'd expect for such a derivative:
\begin{itemize}
\item We know that the integral of this derivative must be equal to 1 to be equivalent to $u(t)$.
\item The derivative must be zero for $t < 0$ since $u(t)$ is constant for $t < 0$, and the derivative must also be zero for $t > 0$ since $u(t)$ is constant for $t > 0$ - the only point at which $u(t)$ is not constant is at $t = 0$.
\end{itemize}

This "function" is called the \textbf{Dirac delta function} $\delta(t)$, and it is defined as follows:
\[
\delta(t) =
\begin{cases}
\infty, \quad t = 0 \\
0, \quad t \neq 0
\end{cases}
\]

In the example above, we have that $q(t) = 1000 + 2000\delta(t - 1)$.

We could even consider a case where at $t = 1$, we deposit \$1000, and at $t = 2$, we withdraw \$500. In this case, we have that $q(t) = 1000\delta(t - 1) - 500\delta(t - 2)$.

In particular, we visualize the Dirac delta function like this:

\begin{figure}[H]
    \centering
    \includegraphics[width=0.5\textwidth]{images/laplace_transforms/dirac_delta_function.png}
    \caption{Graph of the Dirac delta function}
    \label{fig:dirac_delta}
\end{figure}

As convention, we write the $a$ beside the delta function to indicate the integral of the delta function:
\[
\int_{-\infty}^{\infty} a \delta(t) dt = a
\]

Now recall piecewise smooth functions. These functions are broken up into smooth pieces that have limitson the right and on th eleft that exist at each break. These are therefore "nice" functions that switch on or off with different applications of $u(t)$.

For example, consider the function:
\[
f(t) =
\begin{cases}
t, \quad t < 1 \\
1-\sqrt{t}, \quad 1 < t < 2 \\
t-2, \quad t > 2
\end{cases}
\]

This function appears like this:

\begin{figure}[H]
    \centering
    \includegraphics[width=0.5\textwidth]{images/laplace_transforms/piecewise_smooth_function.png}
    \caption{Graph of a piecewise smooth function}
    \label{fig:piecewise_smooth}
\end{figure}

We can write this function in terms of the Heaviside function as follows:
\[
f(t) = t + (1 - \sqrt{t} - t)u(t - 1) + (t - 2 - (1 - \sqrt{t}))u(t - 2)
\]

In general, we can "turn on" a function $f(t)$ at $t = a$ and "turn off" the function at $t = b$ by writing $f(t)(u(t - a) - u(t - b))$. We can also "turn on" a function at $t = a$ and leave it on for all $t > a$ by writing $f(t)u(t - a)$. Then for a piecewise smooth function, we can write it as a sum of functions that are "turned on" at different points. For example, if we have a piecewise smooth function with break points at $t = a_1, a_2, \ldots, a_n$, then we can write the function as:
\[
f(t) = f_1(t) + (f_2(t) - f_1(t))u(t - a_1) + (f_3(t) - f_2(t))u(t - a_2) + \ldots + (f_n(t) - f_{n-1}(t))u(t - a_{n-1})
\]

Basically, at $a_1$, we "turn on" the function $f_2(t)$ with multiplication by $u(t - a_1)$ while the subtraction of $f_1(t) u(t - a_1)$ at the same time "turns off" the function $f_1(t)$, and this process continues for each break point.

\dfn{Generalized derivative}
{
The \textbf{generalized derivative} of a function $f(t)$ is defined as the limit of the difference quotient as $h$ approaches zero, where the difference quotient is given by:
\[
\frac{f(t + h) - f(t)}{h}
\]
This definition allows us to extend the concept of differentiation to functions that may not be differentiable in the traditional sense, such as piecewise smooth functions or functions with discontinuities. The generalized derivative can be used to analyze the behavior of such functions and to solve differential equations that involve them.
}

In particular, given a piecewise smooth function $Q(t)$, we can now find a derivative at point $a$ by:
\[
\dot{Q}(t) = (Q(a^+) - Q(a^-))\delta(t - a)
\]
where $Q(a^+)$ and $Q(a^-)$ are the right and left limits of $Q(t)$ at $t = a$, respectively.

We denote this as:
\[
\boxed{[Q]_a \delta (t - a)}
\]

\ex{Example}{
Consider the function:
\[
Q(t) =
\begin{cases}
0, \quad t < 1 \\
1000t, \quad 1 < t < 2 \\
2000, \quad t > 2
\end{cases}
\]

Find $\dot{Q}(t)$.
}
\sol{
We can write $Q(t)$ in terms of the Heaviside function as follows:
\begin{align*}
Q(t) = 1000t u(t - 1) + (2000 - 1000t)u(t - 2)
= 1000t u(t - 1) + 2000u(t - 2) - 1000t u(t - 2)
\end{align*}

We have three terms to differentiate. 

\begin{align*}
\frac{d}{dt} 1000t u(t - 1) &= 1000u(t - 1) + 1000t \delta(t - 1) \\
\frac{d}{dt} 2000u(t - 2) &= 2000 \delta(t - 2) \\
\frac{d}{dt} 1000t u(t - 2) &= 1000u(t - 2) + 1000t \delta(t - 2)
\end{align*}

Now we can combine these three terms to get:
\begin{align*}
\dot{Q}(t) &= 1000u(t - 1) + 1000t \delta(t - 1) + 2000 \delta(t - 2) - 1000u(t - 2) - 1000t \delta(t - 2) \\
&= 1000u(t - 1) - 1000u(t - 2) + 1000t \delta(t - 1) + (2000 - 1000t) \delta(t - 2)
\end{align*}

Now we use the property that $f(t) = f(a) \delta(t - a)$ to simplify the delta function terms:
\begin{align*}
\dot{Q}(t) &= 1000u(t - 1) - 1000u(t - 2) + 1000t \delta(t - 1) + (2000 - 1000t) \delta(t - 2) \\
&= 1000u(t - 1) - 1000u(t - 2) + 1000 \cdot 1 \cdot \delta(t - 1) + (2000 - 1000 \cdot 2) \delta(t - 2) \\
&= 1000u(t - 1) - 1000u(t - 2) + 1000 \delta(t - 1) + 0 \cdot \delta(t - 2) \\
&= 1000u(t - 1) - 1000u(t - 2) + 1000 \delta(t - 1)
\end{align*}

This can be interpeted as follows: at $t = 1$, we have a deposit of \$1000, which is represented by the delta function term. For $t > 1$, we have a continuous deposit of \$1000 per year, which is represented by the first Heaviside function term. Finally, at $t = 2$, we stop depositing money, which is represented by the second Heaviside function term.
}

Finally, we note that we can shift the delta function as follows:
\begin{equation*}
\delta(t - a) = \frac{d}{dt} u(t - a) 
\end{equation*}

And recalling that:
\begin{equation*}
\int_{-\infty}^{\infty} \delta(t - 1) dt = \int_{-\infty}^{\infty} \delta(t') dt' = 1
\end{equation*}

Applying this to the integral of the delta function multiplied by some function $f(t)$, we have:
\begin{align*}
\int_{-\infty}^{\infty} f(t') \delta (t' - t) dt' &= f(t) \int_{-\infty}^{\infty} \delta(t' - t) dt' \\ 
\end{align*}

We can do this because the delta function is zero everywhere except at $t' = t$, so we can factor out $f(t)$ from the integral. It effectively has a single value, and hence can be treated as a constant. Now, since the integral of the delta function is 1, we have:
\begin{align*}
\int_{-\infty}^{\infty} f(t') \delta (t' - t) dt' &= f(t) \cdot 1 \\
&= f(t)
\end{align*}

\section{Unit response function}
\subsection{First-order linear systems}

Let there be a system described by:
\[
\dot{x} + kx = r \cdot u(t)
\]

That is, before $t = 0$, the system is at rest, and at $t = 0$, we "turn on" the system with a constant input of $r$. We want to find the response of the system to this input.

This can easily be solved using the integrating factor method with $\mu(t) = e^{kt}$, which gives us:
\begin{align*}
e^{kt} \dot{x} + ke^{kt} x &= re^{kt} u(t) \\
\frac{d}{dt} (e^{kt} x) &= re^{kt} u(t) \\
e^{kt} x &= r \int_{0}^{t} e^{k\overline{t}} d\overline{t} \\
x &= re^{-kt} \int_{0}^{t} e^{k\overline{t}} d\overline{t} \\
&= re^{-kt} \cdot \frac{e^{kt} - 1}{k} \\
&= \frac{r}{k} (1 - e^{-kt}) \quad \text{for } t > 0
\end{align*}


\dfn{Unit impulse response function}
{
    The \textbf{unit impulse response function} of a system is the response of the system to a unit impulse input ($\delta(t)$).
}

 For a first-order linear system described by the equation $\dot{x} + kx = r \cdot u(t)$, the unit response function is given by:
\begin{equation*}
\boxed{
v(t) = \frac{r}{k} (1 - e^{-kt}) u(t)
}
\end{equation*}

We now plug $v(t)$ into the original equation (letting $r = 1$):
\[
\dot{v} + kv = u(t) \quad \text{w/} \quad v(0^-) = 0
\]

We can now solve for $\dot{v}$:
\begin{align*}
\dot{v} &= u(t) - kv \\
&= u(t) - k \cdot \frac{1}{k} (1 - e^{-kt}) u(t) \\
&= u(t) - (1 - e^{-kt}) u(t) \\
&= e^{-kt} u(t)
\end{align*}

Note that at $t = 0^+$, we have:
\begin{align*}
\dot{v}(0^+) &= e^{-k \cdot 0} u(0) \\
&= 1 \cdot 1 \\
&= 1
\end{align*}

And we know that $u(0^+) = 1$, so basically we have that:
\begin{equation*}
\boxed{
\dot{v}(0^+) = u(0^+)}
\end{equation*}

\ex{Basic unit impulse response function}{
Consider the system described by the following equation:
\[
\dot{x} + kx = \delta (t)
\]
Find the response of the system to this input.
}
\sol{
Solving the homogeneous equation $\dot{x} + kx = 0$ gives us that $x_h(t) = Ce^{-kt}$. Now we can use the method of variation of parameters to find a particular solution. Let $x_p(t) = C(t)e^{-kt}$, then we have:
\begin{align*}
\dot{x}_p + kx_p &= \dot{C}e^{-kt} - kCe^{-kt} + kCe^{-kt} \\
&= \dot{C}e^{-kt} \\
&= \delta(t)
\end{align*}
Now we can solve for $\dot{C}$:
\begin{align*}
\dot{C} &= e^{kt} \delta(t) \\
&= e^{k \cdot 0} \delta(t) \\
&= 1 \cdot \delta(t) \\
&= \delta(t)
\end{align*}
Now we can integrate both sides to find $C(t)$:
\begin{align*}
C(t) &= \int_{-\infty}^{t} \delta(\overline{t}) d\overline{t} \\
&= u(t)
\end{align*}
Finally, we can write the particular solution as follows:
\begin{align*}
x_p(t) &= C(t)e^{-kt} \\
&= u(t)e^{-kt}
\end{align*}
Thus, the general solution to the nonhomogeneous equation is given by:
\begin{align*}
x(t) &= x_h(t) + x_p(t) \\
&= Ce^{-kt} + u(t)e^{-kt}
\end{align*}
Now we can apply the initial condition $x(0^-) = 0$ to solve for $C$:
\begin{align*}
x(0^-) &= Ce^{-k \cdot 0} + u(0^-)e^{-k \cdot 0} \\
&= C \cdot 1 + 0 \cdot 1 \\
&= C
\end{align*}
Thus, we have that $C = 0$, and the response of the system to the input $\delta(t)$ is given by:
\begin{equation*}
\boxed{
x(t) = u(t)e^{-kt}}
\end{equation*}
}

We can further characterize the unit impulse function. For example, since the function is only "on" from $t >0$, we have that $w(0^-) = 0$ and $w(0^+) = 1$.

Then, because $\delta(0^+) = 0$, we can also find that:
\begin{align*}
\dot{w}(0^+) &= \frac{d}{dt} w(t)  \bigg|_{t=0^+} \\
&= \frac{d}{dt} (u(t)e^{-kt}) \bigg|_{t=0^+} \\
&= -ke^{-kt}u(t) + e^{-kt} \delta(t) \bigg|_{t=0^+} \\
&= -k \cdot 1 \cdot 1 + 1 \cdot 0 \\
&= -k
\end{align*}

\subsection{Second-order linear systems}
Consider the second-order linear system described by:
\[
m \ddot{x} + kx = \delta (t)
\]

That is, there is no damping in the system, and an increase in momentum occurs at $t = 0$ with no change in position.

We will use the \textbf{shrinking interval method} to solve this problem. We will first solve the homogeneous equation $m \ddot{x} + kx = 0$ to find the general solution, and then we will apply the initial conditions to find the particular solution.

Solving the homogeneous equation:
\begin{align*}
m r^2 + k &= 0 \\
r^2 &= -\frac{k}{m} \\
r &= \pm i \sqrt{\frac{k}{m}} \quad \text{(let } \omega_m = \sqrt{\frac{k}{m}} \text{)} \\
r &= \pm i \omega_m
\end{align*}

Then writing $x_h(t)$:
\begin{align*}
x_h(t) = c_1 \cos (\omega_m t) + c_2 \sin (\omega_m t)
\end{align*}

To find the initial conditions, consider the behaviour of the system as $t$ approaches zero from the left and the right.

As $t$ approaches zero from the left, we have that $x(0^-) = 0$ and $\dot{x}(0^-) = 0$ since the system is at rest before $t = 0$.

We can integrate the equation $m \ddot{x} + kx = \delta (t)$ from $t = -\epsilon$ to $t = \epsilon$ for some small $\epsilon > 0$:
\begin{align*}
\int_{-\epsilon}^{\epsilon} m \ddot{x} + kx dt &= \int_{-\epsilon}^{\epsilon} \delta (t) dt \\
m \dot{x}(\epsilon) - m \dot{x}(-\epsilon) + k \int_{-\epsilon}^{\epsilon} x dt &= 1
\end{align*}

Now, as $\epsilon$ approaches zero, we have that $\dot{x}(-\epsilon) \to 0$ and $\int_{-\epsilon}^{\epsilon} x dt \to 0$, so we can simplify the equation to get:
\begin{align*}
m \dot{x}(0^+) &= 1 \\
\dot{x}(0^+) &= \frac{1}{m}
\end{align*}

Now we can apply the initial conditions to the general solution to find the particular solution:
\begin{align*}
x(0^+) &= c_1 \cos (0) + c_2 \sin (0) \\
&= c_1 \\
\end{align*}

\begin{align*}
\dot{x}(0^+) &= -c_1 \omega_m \sin (0) + c_2 \omega_m \cos (0) \\
&= c_2 \omega_m \\
&= \frac{1}{m} \\
c_2 &= \frac{1}{m \omega_m}
\end{align*}

Thus, the response of the system to the input $\delta(t)$ is given by:
\begin{equation*}
\boxed{
x(t) = \frac{1}{m \omega_m} \sin (\omega_m t) u(t)}
\end{equation*}

\subsection{Generalization to $n$-th order linear systems}

Consider the $n$-th order linear system described by:
\[
a_n x^{(n)} + a_{n-1} x^{(n-1)} + \ldots + a_1 \dot{x} + a_0 x = u (t)
\]

We know that for $t < 0$, we have that $x(0^-) = 0$, $\dot{x}(0^-) = 0$, $\ldots$, $x^{(n-1)}(0^-) = 0$ since the system is at rest before $t = 0$.

Applying the shrinking interval method, we can integrate the equation from $t = -\epsilon$ to $t = \epsilon$ for some small $\epsilon > 0$:
\begin{align*}
\int_{-\epsilon}^{\epsilon} a_n x^{(n)} + a_{n-1} x^{(n-1)} + \ldots + a_1 \dot{x} + a_0 x dt &= \int_{-\epsilon}^{\epsilon} u (t) dt \\
a_n x^{(n-1)}(\epsilon) - a_n x^{(n-1)}(-\epsilon) + a_{n-1} x^{(n-2)}(\epsilon) - a_{n-1} x^{(n-2)}(-\epsilon) + \ldots + a_1 x(\epsilon) - a_1 x(-\epsilon) + a_0 \int_{-\epsilon}^{\epsilon} x dt &= 1
\end{align*}

Then all terms with $x^{(n-1)}(-\epsilon)$, $x^{(n-2)}(-\epsilon)$, $\ldots$, $x(-\epsilon)$, and $\int_{-\epsilon}^{\epsilon} x dt$ will approach zero as $\epsilon$ approaches zero, so we can simplify the equation to get:
\begin{align*}
a_n x^{(n-1)}(0^+) + a_{n-1} x^{(n-2)}(0^+) + \ldots + a_1 x(0^+) &= 1 \\
a_n x^{(n-1)}(0^+) &= 1 \\
x^{(n-1)}(0^+) &= \frac{1}{a_n}
\end{align*}

So we have the following initial conditions:
\begin{itemize}
    \item $x(0^+) = 0$
    \item $\dot{x}(0^+) = 0$
    \item $\ldots$
    \item $x^{(n-2)}(0^+) = 0$
    \item $x^{(n-1)}(0^+) = \frac{1}{a_n}$
\end{itemize}

\subsection{Unit step response function}

\dfn{Unit step response function}
{
    The \textbf{unit step response function} of a system is the response of the system to a unit step input ($u(t)$).
}

\ex{Finding the unit step response function}{
Find the unit step response function of the system described by the following equation:
\[
m\ddot{x} + kx = u(t)
\]
}
\sol{
We know that the homogeneous solution is:
\begin{align*}
x_h(t) = c_1 \cos (\omega_m t) + c_2 \sin (\omega_m t)
\end{align*}

To find the particular solution where $t>0$ implies $u(t) = 1$, we know that $x(0^+) = 0$, but we need the behaviour at $x(0^+)$ to find the particular solution. We can integrate the equation $m\ddot{x} + kx = u(t)$ from $t = -\epsilon$ to $t = \epsilon$ for some small $\epsilon > 0$:
\begin{align*}
\int_{-\epsilon}^{\epsilon} m\ddot{x} + kx dt &= \int_{-\epsilon}^{\epsilon} u(t) dt \\
m \dot{x}(\epsilon) - m \dot{x}(-\epsilon) + k \int_{-\epsilon}^{\epsilon} x dt &= 0
\end{align*}
Now, as $\epsilon$ approaches zero, we have that $\dot{x}(-\epsilon) \to 0$ and $\int_{-\epsilon}^{\epsilon} x dt \to 0$, so we can simplify the equation to get:
\begin{align*}
m \dot{x}(0^+) &= 0 \\
\dot{x}(0^+) &= 0
\end{align*}
Then computing coefficients, we find that:
\begin{align*}
c_1 &= \frac{-1}{k} \\
c_2 &= 0
\end{align*}

Then we get:
\begin{align*}
x(t) = \frac{1}{k}\cos (\omega_m t) - \frac{1}{k} \quad \text{for } t > 0
\end{align*}

This function is only non-zero when $t > 0$, so we can multiply the function by $u(t)$ to get the final solution.

Hence the \textbf{unit step response function} of the system is given by:
\begin{equation*}
\boxed{
x(t) = \frac{1}{k} (1 - \cos (\omega_m t)) u(t)
}
\end{equation*}

}

Recall that we can write the system as:
\[
p(D) x = q(t)
\]
where $p(D) = a_n D^n + a_{n-1} D^{n-1} + \ldots + a_1 D + a_0$ and $q(t)$ is some input function.

Then, differentiating both sides once:
\begin{align*}
p(D) \dot{x} &= \dot{q}(t)
\end{align*}

Now for the unit step function system:
\begin{align*}
p(D) x &= u(t)
\end{align*}

We can differentiate both sides once to get:
\begin{align*}
p(D) \dot{x} &= \delta(t)
\end{align*}

Thus, we can see that the unit step response function is the same as the unit impulse response function, except that the initial conditions are shifted by one order. For example, for a first-order linear system, the unit step response function will have an initial condition of $\dot{x}(0^+) = 0$ instead of $\dot{x}(0^+) = \frac{1}{a_1}$.

Also, given the unit step response function $v(t)$, we can find the unit impulse response function $w(t)$ by differentiating $v(t)$:
\begin{align*}
\boxed{
\dot{v}(t) = w(t)
}
\end{align*}

\section{Convolution}
Let there be some system:
\[
p(D) x = q(t)
\]

where $q(t)$ is some input function, which is made of a sum of impulses. We can write $q(t)$ as a sum of impulses as follows:
\begin{align*}
q(t) &= \int_{-\infty}^{\infty} q(\tau) \delta(t - \tau) d\tau
\end{align*}

Recall the unit impulse response function $w(t)$. We can also write using linearity and $\delta$ function properties that:
\begin{align*}
p(D) a w(t - \tau) &= a \delta(t - \tau)
\end{align*}

Using the linearity of the system (and the fact that we can shift the delta function), we can write the response of the system to the input $q(t)$ as follows:
\begin{align*}
x(t) &= \sum_{-\infty}^{\infty} q(\tau) w(t - \tau) \\
&= \int_{-\infty}^{\infty} q(\tau) w(t - \tau) d\tau
\end{align*}

This is allowed as $q(\tau)$ is just a constant with respect to $t$.

We see now that we can obtain any response of the system by mixing the input function $q(t)$ with the unit impulse response function $w(t)$. 


\dfn{Convolution}
{
    The \textbf{convolution} of two functions $f(t)$ and $g(t)$ is defined as the integral of the product of the two functions, where one function is reversed and shifted. Mathematically, it is given by:
    \[
    (f * g)(t) = \int_{-\infty}^{\infty} f(\tau) g(t - \tau) d\tau
    \]
}

Since integrals are linear operators, we also have:
\begin{align*}
\frac{d}{dt} (u \* v) = (p(D) u) \* v = u \* (p(D) v)
\end{align*}

We also recall that we found it possible to "extract" the value of a function at a point by multiplying the function by a delta function and integrating, i.e.
\begin{align*}
\int_{-\infty}^{\infty} f(t) \delta(t - a) dt = f(a)
\end{align*}

We now saw that this was the convolution of $f(t)$ with $\delta(t)$.

Now, we can write the response of the system to the input $q(t)$ as the convolution of $q(t)$ with the unit impulse response function $w(t)$:
\begin{align*}
x(t) &= \int_{-\infty}^{\infty} q(\tau) w(t - \tau) d\tau \\
&= (q * w)(t)
\end{align*}

\ex{Using the convolution to solve a system}{
Consider the system described by the following equation:
\[
\dot{x} + 5x = \sin(\omega t) u(t)
\]

Find the general solution to this system.
}

\sol{
We recall that:
\[
w(t) = u(t)e^{-kt} 
\]

In this case, we have that $k = 5$, so we can write the unit impulse response function as follows:
\begin{equation*}
w(t) = u(t)e^{-5t}
\end{equation*}

So find $x(t)$, we simply convolve the input function $\sin(\omega t) u(t)$ with the unit impulse response function $w(t)$. We use complex replacement to find the convolution:
\begin{align*}
x(t) &= \int_{-\infty}^{\infty} \sin(\omega \tau) u(\tau) w(t - \tau) d\tau \\
&= \int_{0}^{t} \sin(\omega \tau) e^{-5(t - \tau)} d\tau \\
&= e^{-5t} \int_{0}^{t} \sin(\omega \tau) e^{5\tau} d\tau \\
&= e^{-5t} \cdot \frac{e^{5t}(5\sin(\omega t) - \omega \cos(\omega t)) + \omega}{25 + \omega^2} \\
&= \frac{5\sin(\omega t) - \omega \cos(\omega t)}{25 + \omega^2} + \frac{\omega e^{-5t}}{25 + \omega^2}
\end{align*}

where $\omega \neq 5$. Notably, we have extracted both the particular solution (which is the second term) and the homogeneous solution (which is the first term) from the convolution. However, this formula does not work when resonance occurs, i.e. when $\omega = 5$.

}

\subsection{Properties of convolution:}
\begin{itemize}
\item Convolution is commutative: $f * g = g * f$
\item Convolution is associative: $f * (g * h) = (f * g) * h$
\item Convolution is distributive over addition: $f * (g + h) = f * g + f * h$
\item Convolution with a delta function: $f * \delta = f$
\item Convolution with a shifted delta function: $f * \delta(t - a) = f(t - a)$
\item Convolution with a unit step function: $f * u = \int_{-\infty}^{t} f(\tau) d\tau$
\item Convolution with an exponential function: $f * e^{at} = \int_{-\infty}^{t} f(\tau) e^{a(t - \tau)} d\tau$
\end{itemize}

\section{Laplace transforms}
The Laplace transform is a generalization of the above. Looking at the convolution product to get $x(t)$:
\begin{equation*}
x(t) = \int_{-\infty}^{\infty} q(\tau) w(t - \tau) d\tau
\end{equation*}

We expand out $w(t - \tau)$ (for a first-order case, and assuming $t > 0$):
\begin{align*}
x(t) &= \int_{-\infty}^{\infty} q(\tau) u(t - \tau) e^{-k(t - \tau)} d\tau 
\end{align*}

We can take out the $e^{-kt}$ term since it is a constant with respect to $\tau$:
\begin{align*}
x(t) &= e^{-kt} \int_{-\infty}^{\infty} q(\tau) u(t - \tau) e^{k\tau} d\tau
\end{align*}

Note that $u(t - \tau)$ is zero for $\tau > t$ and $q(\tau) = 0$ for $\tau < 0$, so we can change the limits of integration to get:
\begin{align*}
x(t) &= e^{-kt} \int_{0}^{t} q(\tau) e^{k\tau} d\tau
\end{align*}

Now we make the substitution $s = -k$ to get:
\begin{align*}
x(t) &= e^{st} \int_{0}^{t} q(\tau) e^{-s\tau} d\tau
\end{align*}

Now we can define the Laplace transform of $q(t)$ as follows:
\begin{equation*}
\boxed{
\mathcal{L}\{q(t)\} = \int_{0}^{\infty} q(\tau) e^{-s\tau} d\tau
}
\end{equation*}

Then we can write $x(t)$ in terms of the Laplace transform of $q(t)$ as follows:
\begin{align*}
x(t) &= e^{st} \int_{0}^{t} q(\tau)
e^{-s\tau} d\tau \\
&= e^{st} \mathcal{L}\{q(t)\}
\end{align*}

\dfn{Laplace transform}
{
    The \textbf{Laplace transform} of a function $f(t)$ is defined as the integral of the product of the function and an exponential decay factor, given by:
    \[
    \mathcal{L}\{f(t)\} = \int_{0}^{\infty} f(t) e^{-st} dt
    \]
    where $s$ is a complex number. 
}

Recall that we found that $x(t) = (q * w)(t)$, where $w(t)$ is the unit impulse response function. When we take the Laplace transform of both sides, we get:
\begin{align*}
\mathcal{L}\{x(t)\} &= \mathcal{L}\{(q * w)(t)\} \\
&= \mathcal{L}\{q(t)\} \cdot \mathcal{L}\{w(t)\}
\end{align*}

\ex{Laplace transform of 1}{
Find the Laplace transform of the function $f(t) = 1$.
}
\sol{
We can compute the Laplace transform of $f(t) = 1$ as follows:
\begin{align*}
\mathcal{L}\{1\} &= \int_{0}^{\infty} 1 \cdot e^{-st} dt \\
&= \int_{0}^{\infty} e^{-st} dt \\
&= \left[ \frac{e^{-st}}{-s} \right]_{0}^{\infty} \\
&= \frac{1}{s}
\end{align*}
Thus, we have that:
\begin{equation*}
\boxed{
\mathcal{L}\{1\} = \frac{1}{s}}
\end{equation*}
}

\nt{
    We know that $s \in \bbC$. Taking the modulus of $e^{st}$, we have:
    \begin{align*}
    |e^{st}| &= | e^{(a+bi)t} | \\
    &= | e^{at} \cdot e^{ibt} | \\
    &= | e^{at} | \cdot | e^{ibt} | \\
    &= e^{at} \cdot 1 \\
    &= e^{at}
    \end{align*}
    So the Laplace transform of $f(t) = 1$ converges if and only if $a < 0$, i.e. $\Re(s) < 0$.
}

\ex{Laplace transform of $t^n$}{
Find the Laplace transform of the function $f(t) = t^n$, where $n$ is a positive integer.
}
\sol{
We can compute the Laplace transform of $f(t) = t^n$ as follows:
\begin{align*}
\mathcal{L}\{t^n\} &= \int_{0}^{\infty} t^n e^{-st} dt \\
&= \left[ -\frac{t^n e^{-st}}{s} \right]_{0}^{\infty} + \frac{n}{s} \int_{0}^{\infty} t^{n-1} e^{-st} dt \\
&= 0 + \frac{n}{s} \mathcal{L}\{t^{n-1}\}
\end{align*}
Now we can apply this process iteratively to find the Laplace transform of $t^n$:
\begin{align*}
\mathcal{L}\{t^n\} &= \frac{n}{s} \mathcal{L}\{t^{n-1}\} \\
\mathcal{L}\{t^n\} &= \frac{n}{s} \mathcal{L}\{t^{n-1}\} \\
&= \frac{n}{s} \cdot \frac{n-1}{s} \mathcal{L}\{t^{n-2}\} \\
&= \frac{n}{s} \cdot \frac{n-1}{s} \cdot \frac{n-2}{s} \mathcal{L}\{t^{n-3}\} \\
&\vdots \\
&= \frac{n(n-1)(n-2)\cdots 1}{s^n} \mathcal{L}\{t^0\} \\
&= \frac{n!}{s^n} \mathcal{L}\{1\}
\end{align*}
It remains to find $\mathcal{L}\{1\}$:
\begin{align*}
\mathcal{L}\{1\} &= \int_{0}^{\infty} e^{-st} \, dt 
= \left[ -\frac{e^{-st}}{s} \right]_{0}^{\infty} 
= \frac{1}{s}
\end{align*}
Therefore:
\begin{align*}
\mathcal{L}\{t^n\} &= \frac{n!}{s^n} \cdot \frac{1}{s} = \frac{n!}{s^{n+1}}
\end{align*}
}

Laplace transforms also satisfy linearity, i.e. for any constants $a$ and $b$, we have:
\begin{align*}
\mathcal{L}\{af(t) + bg(t)\} &= a \mathcal{L}\{f(t)\} + b \mathcal{L}\{g(t)\}
\end{align*}

They also satisfy shifting properties, i.e. for any constant $a$, we have:
\begin{align*}
\mathcal{L}\{e^{at} f(t)\} &= F(s - a) \\
\mathcal{L}\{f(t - a) u(t - a)\} &= e^{-as} F(s)
\end{align*}

\section{Using Laplace transforms to solve linear systems}
Taking the Laplace transform of a derivative:
\begin{align*}
\mathcal{L}\{\dot{x}(t)\} &= \int_{0}^{\infty} \dot{x}(t) e^{-st} dt \\
&= \left[ x(t) e^{-st} \right]_{0}^{\infty} + s \int_{0}^{\infty} x(t) e^{-st} dt \\
&= 0 - x(0) + sX(s) \\
&= sX(s) - x(0)
\end{align*}

Taking the Laplace transform of a second derivative:
\begin{align*}
\mathcal{L}\{\ddot{x}(t)\} &= \int_{0}^{\infty} \ddot{x}(t) e^{-st} dt \\
&= \left[ \dot{x}(t) e^{-st} \right]_{0}^{\infty} + s \int_{0}^{\infty} \dot{x}(t) e^{-st} dt \\
&= 0 - \dot{x}(0) + s(sX(s) - x(0)) \\
&= s^2 X(s) - sx(0) - \dot{x}(0)
\end{align*}

So we see that the Laplace transform of a derivative is a polynomial in $s$ multiplied by the Laplace transform of the original function, minus some terms involving the initial conditions.

\ex{Using Laplace transforms to solve a system}{
Consider the system described by the following equation:
\[
\dot{w} + 3w = \delta (t)
\]

with initial rest conditions (i.e. $w(0) = 0$ and $\dot{w}(0) = 0$).

Find the response of the system to this input.
}
\sol{
Taking the Laplace transform of both sides of the equation, we get:
\begin{align*}
sW(s) - w(0) + 3W(s) &= 1 \\
(s + 3)W(s) &= 1 \\
W(s) &= \frac{1}{s + 3}
\end{align*}

Note that $p(s) = s+3$, so $W(s) = \frac{1}{p(s)}$. 

We need to find some function $w(t)$ such that $\mathcal{L}\{w(t)\} = \frac{1}{s + 3}$. We can use the shifting property of Laplace transforms to find $w(t)$:
\begin{align*}
\mathcal{L}\{e^{-3t}\} &= \frac{1}{s - (-3)} \\
&= \frac{1}{s + 3}
\end{align*}

Thus, we have that:
\begin{equation*}
\boxed{
w(t) = e^{-3t} u(t)}
\end{equation*}
}

Interestingly, this demonstrates that for the unit impulse response function, we can extract $p(s)$ from $W(s)$.

\ex{}{
Consider the system described by the following equation:
\[
\dot{x} + 3x = e^{-t}, \quad x(0^+) = 0
\]
}
\sol{
Taking the Laplace transform of both sides of the equation, we get:
\begin{align*}
sX(s) - x(0) + 3X(s) &= \frac{1}{s + 1} \\
(s + 3)X(s) &= \frac{1}{s + 1} \\
X(s) &= \frac{1}{(s + 3)(s + 1)}
\end{align*}

To find $x(t)$, we can use partial fraction decomposition to write $X(s)$ as a sum of simpler fractions:
\begin{align*}
X(s) &= \frac{1}{(s + 3)(s + 1)} \\
&= \frac{A}{s + 3} + \frac{B}{s + 1}
\end{align*}

Solving for $A$ and $B$, we get:
\begin{align*}
A(s + 1) + B(s + 3) &= 1 \\
A = \frac{1}{2}, \quad B = -\frac{1}{2}
\end{align*}

Then, we can write $X(s)$ as follows:
\begin{align*}
X(s) &= \frac{1/2}{s + 3} - \frac{1/2}{s + 1}
\end{align*}

We can use $e^{at}$ to find the inverse Laplace transform of each term:
\begin{align*}
\mathcal{L}\{e^{-3t}\} &= \frac{1}{s + 3} \\
\mathcal{L}\{e^{-t}\} &= \frac{1}{s + 1}
\end{align*}

Thus, we have that:
\begin{equation*}
\boxed{
x(t) = \frac{1}{2} e^{-3t} u(t) - \frac{1}{2} e^{-t} u(t)
}
\end{equation*}

We can also get both $x_h(t)$ and $x_p(t)$ from this. In particular, the homogeneous solution is the term with $e^{-3t}$, and the particular solution is the term with $e^{-t}$.
}

\ex{}{
Consider the system described by the following equation:
\[
\dot{x} + 3x = e^{-t}, \quad x(0^+) = 5
\]
}
\sol{
We interpret this as the system being at rest before $t = 0$, and then at $t = 0$, the system is given an initial condition of $x(0^+) = 5$.

So we rewrite as:
\begin{align*}
\dot{x} + 3x &= e^{-t} + 5\delta(t)
\end{align*}

Then we can take the Laplace transform of both sides to get:
\begin{align*}
sX(s) - x(0) + 3X(s) &= \frac{1}{s + 1} + 5 \\
(s + 3)X(s) &= \frac{1}{s + 1} + 5 \\
X(s) &= \frac{1}{(s + 3)(s + 1)} + \frac{5}{s + 3}
\end{align*}

Then we can use partial fraction decomposition to find the inverse Laplace transform of the first term, and we can use $e^{at}$ to find the inverse Laplace transform of the second term:
\begin{align*}
\mathcal{L}\{e^{-3t}\} &= \frac{1}{s + 3} \\
\mathcal{L}\{e^{-t}\} &= \frac{1}{s + 1}
\end{align*}

Thus, we have that:
\begin{equation*}
\boxed{
x(t) = \frac{1}{2} e^{-3t} u(t) - \frac{1}{2} e^{-t} u(t) + 5 e^{-3t} u(t)
}
\end{equation*}
}

\ex{$n$-th order system}{
    Consider a function $f(t)$ such that $\mathcal{L}\{f(t)\} = F(s)$. Find the Laplace transform of $f^{(n)}(t)$, where $f^{(n)}(t)$ is the $n$-th derivative of $f(t)$.
}
\sol{
We can compute the Laplace transform of $f^{(n)}(t)$ as follows:
\begin{align*}
\mathcal{L}\{f^{(n)}(t)\} &= \int_{0}^{\infty} f^{(n)}(t) e^{-st} dt \\
&= \left[ f^{(n-1)}(t) e^{-st} \right]_{0}^{\infty} + s \int_{0}^{\infty} f^{(n-1)}(t) e^{-st} dt \\
&= 0 - f^{(n-1)}(0) + s \mathcal{L}\{f^{(n-1)}(t)\} \\
&= s \mathcal{L}\{f^{(n-1)}(t)\} - f^{(n-1)}(0)
\end{align*}
Continuing this process iteratively, we get:
\begin{align*}
\mathcal{L}\{f^{(n)}(t)\} &= s \mathcal{L}\{f^{(n-1)}(t)\} - f^{(n-1)}(0) \\
&= s(s \mathcal{L}\{f^{(n-2)}(t)\} - f^{(n-2)}(0)) - f^{(n-1)}(0) \\
&= s^2 \mathcal{L}\{f^{(n-2)}(t)\} - sf^{(n-2)}(0) - f^{(n-1)}(0) \\
&\vdots \\
&= s^n \mathcal{L}\{f(t)\} - \sum_{k=1}^{n} s^{n-k} f^{(k-1)}(0) \\
&= s^n F(s) - \sum_{k=1}^{n} s^{n-k} f^{(k-1)}(0)
\end{align*}

Read this as $s^n F(s)$ minus a sum of terms involving the initial conditions of $f(t)$ and its derivatives up to order $n-1$.
}

\ex{Solving a second-order system}{
    Consider the following equation:
\[
\ddot{w} + 2 \dot{w} + 2w = \delta (t)
\]

Find the impulse response function of this system.
}
\sol{
Taking the Laplace transform of both sides of the equation, we get:
\begin{align*}
s^2 W(s) - sw(0) - \dot{w}(0) + 2(sW(s) - w(0)) + 2W(s) &= 1 \\
(s^2 + 2s + 2)W(s) &= 1 \\
W(s) &= \frac{1}{s^2 + 2s + 2}
\end{align*}
To find $w(t)$, we can complete the square in the denominator of $W(s)$:
\begin{align*}
W(s) &= \frac{1}{(s + 1)^2 + 1}
\end{align*}
First, from the Laplace table, we see a $s^2$ term indicating a sinusoidal function. Then we can use the shifting property of Laplace transforms to find $w(t)$:
\begin{align*}
\mathcal{L}\{e^{-t} \sin t\} &= \frac{1}{(s + 1)^2 + 1}
\end{align*}
Thus, we have that:
\begin{equation*}
\boxed{
w(t) = e^{-t} \sin t \cdot u(t)}
\end{equation*}

Observe the $u(t)$ term, which comes from the fact that the system is at rest before $t = 0$.
}

\dfn{Weight function}{
    The \textbf{weight function} of a system is the response of the system to a unit impulse input ($\delta(t)$).
}

\dfn{Transfer function}{
    The \textbf{transfer function} of a system is the Laplace transform of the weight function of the system.
    }

In particular, the transfer function of $P(D)$ for the system:
\begin{align*}
P(D) x = q(t)
\end{align*}

is given by passing a unit impulse input through the system and taking the Laplace transform of the output. So we have that:
\begin{align*}
p(D) w(t) &= \delta(t) \\
\mathcal{L}\{p(D) w(t)\} &= \mathcal{L}\{\delta(t)\} \\
p(s) W(s) &= 1 \\
W(s) &= \frac{1}{p(s)}
\end{align*}

With this, we can see that to find $x(t)$, we can first find the transfer function $W(s)$, then find the Laplace transform of the input $q(t)$, and then multiply these two together to get $X(s)$, and finally take the inverse Laplace transform to get $x(t)$.

\ex{Using the transfer function to solve a system}{
    Consider the system described by the following equation:
\[
\ddot{x} + 4x = \cos(2t) u(t)
\]

Note that $u(t)$ signals rest initial conditions. Find the response of the system to this input.
}
\sol{
First, we find the transfer function of the system. We can write the system as follows:
\begin{align*}
\ddot{w} + 4w &= \delta(t) \\
p(D) w &= \delta(t)
\end{align*}

Recall that $W(s) = \frac{1}{p(s)}$, so we need to find $p(s)$:
\begin{align*}
p(s) &= s^2 + 4
\end{align*}
Thus, we have that:
\begin{align*}
W(s) &= \frac{1}{s^2 + 4}
\end{align*}

Now, multiplying $W(s)$ by the Laplace transform of the input $\cos(2t) u(t)$, we get:
\begin{align*}
X(s) &= W(s) \cdot \mathcal{L}\{\cos(2t) u(t)\} \\
&= \frac{1}{s^2 + 4} \cdot \frac{s}{s^2 + 4} \\
&= \frac{s}{(s^2 + 4)^2}
\end{align*}

Note this is equivalent to taking the convolution of the input with the weight function of the system, and then taking the Laplace transform of the result.


We now need to find the inverse Laplace transform. Using the table, we can see that the inverse Laplace transform of $\frac{1}{s^2 + 4}$ is $\sin(2t)$. 

However, $X(s)$ looks something like the derivative of $\frac{1}{s^2 + 4}$. In particular:
\begin{align*}
\frac{d}{ds} \left( \frac{1}{s^2 + 4} \right) &= -\frac{2s}{(s^2 + 4)^2} \\
-\frac{1}{2} \cdot \frac{d}{ds} \left( \frac{1}{s^2 + 4} \right) &= \frac{s}{(s^2 + 4)^2}
\end{align*}

So we can use the property of Laplace transforms that states that the Laplace transform of $t f(t)$ is equal to $-F'(s)$, where $F(s)$ is the Laplace transform of $f(t)$. In particular, we have that:
\begin{align*}
\mathcal{L}\{t \sin(2t)\} &= \frac{4s}{(s^2 + 4)^2} \\ \quad \text{ scaling by } \frac{1}{4} \text{ gives } \\
\frac{1}{4} \cdot \frac{4s}{(s^2 + 4)^2} &= \frac{s}{(s^2 + 4)^2}
\end{align*}

Thus, we have that:
\begin{equation*}
\boxed{x(t) = \frac{1}{4} t \sin(2t) \cdot u(t)}
\end{equation*}
}

\section{Poles}
We see that $F(s) = \frac{1}{s}$ has a singularity where $s = 0$, at which the function "blows up". We call such a point a \textbf{pole} of the function. In general, a pole of a function is a point in the complex plane where the function becomes unbounded (i.e., it goes to infinity).

\ex{Poles of a function}{
    Find the poles of the function $F(s) = \frac{1}{s^2 + 4}$.
}
\sol{
The poles of $F(s)$ are the values of $s$ for which the denominator is zero:
\[
s^2 + 4 = 0 \implies s^2 = -4 \implies s = \pm 2i
\]
Thus, the poles are $s = 2i$ and $s = -2i$.
}
